# Benimle Sevgili Olur Musun? ❤️ Bu proje, eğlenceli bir interaktif sevgili olma teklifi oyunudur. Kullanıcı, "Benimle sevgili olur musun?" sorusuna yanıt verirken "Hayır" seçeneği kaçabilir veya "Evet" seçenekleri artabilir. "Evet" seçildiğinde ise sarılan iki minik ayıcık gösterilir ve sevgiliye özel hediye önerisi sunulur. 🎁 ## Özellikler 🚀 - "Hayır" butonu farklı animasyonlarla kullanıcıyı zorlar: - Kaçan buton 🏃‍♂️ - "Evet" butonlarının çoğalması ➕ - "Evet" butonuna tıklanınca animasyonlu ayıcıklar gösterilir 🧸 - Sevgiliye özel affiliate hediye önerisi linki eklenmiştir. ## Kullanım 🛠️ 1. **HTML dosyasını tarayıcınızda açın** veya **bir web hosting servisine yükleyin**. 2. "Benimle sevgili olur musun?" yazısına tıklayın. 3. "Hayır" butonunun nasıl davrandığını keşfedin! 😆 4. "Evet" butonuna basınca sevimli ayıcıkları görün! 🥰 ## Kurulum 🔧 Bu projeyi GitHub Pages, Vercel veya Netlify gibi platformlarla kolayca yayına alabilirsiniz. ## Lisans 📜 Bu proje sadece eğlence amaçlıdır.
